# Download intellij Link for Windows:
    - https://www.jetbrains.com/idea/download/?section=windows
    
    - Scroll down to "IntelliJ IDEA Community Edition"
    
    - Select '.exe' and click download. 


-----------------------------------------------------------------------------------------------
# spring initializr
    - Go to: https://start.spring.io/
    - select these options:
        - Project: Maven
        - Language: Java
        - Spring Boot: 3.2.0
        - Packaging: Jar
        - Java: 17

        Dependencies
            - Spring web
            - Lombok
            - JPA
            - MySQL Driver
            - Spring Boot DevTools

    Note: we can always click on "EXPLORE" button to check out project setup.

    - Click "GENERATE" button.

-----------------------------------------------------------------------------------------------
Configure JRE:
    - File > Project Structure > SDK > click "+" icon > Add JDK
